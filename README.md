git clone https://github.com/tparh/AI-Research-Assistant.git

### 2. Backend Setup

Navigate to the backend directory:

```bash
cd backend
```

Create a virtual environment:

```bash
python -m venv .venv
```

Activate the virtual environment.

**Windows (PowerShell)**

```powershell
.\.venv\Scripts\Activate.ps1
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Create a `.env` file from the example:

```bash
copy .env.example .env
```

Add your Gemini API key to `.env`:

```text
GEMINI_API_KEY=your_api_key_here
```

Start the FastAPI server:

```bash
uvicorn app.main:app --reload
```

The backend runs at:

```text
http://localhost:8000
```

FastAPI documentation:

```text
http://localhost:8000/docs
```

## рџ”„ How the RAG Pipeline Works

1. User uploads a PDF.
2. The backend extracts the document text.
3. Text is divided into smaller chunks.
4. Each chunk is converted into embeddings.
5. Embeddings are stored in ChromaDB.
6. User asks a question.
7. The question is embedded and compared against stored vectors.
8. The most relevant document chunks are retrieved.
9. Gemini generates an answer using the retrieved context.
10. The response is returned to the React frontend.
